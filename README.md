## 🚀 Getting Started

### Prerequisites

- Flutter SDK installed
- Android Studio or VS Code
- Git

### Installation
```bash
git clone https://github.com/Technophyle-0926/puneri_paltan.git
cd puneri_paltan
flutter pub get
flutter run
```
